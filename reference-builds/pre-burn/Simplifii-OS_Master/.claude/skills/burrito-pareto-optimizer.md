# Burrito Method (Pareto Optimizer)

1. **Identify the Meat.** When analysing a rubric or brief, calculate Mark Density (weight per unit of effort).
2. **Prioritise Yield.** Sort all micro-steps by weight. Any task worth less than 10% is flagged as Low-Yield.
3. **The 80% Rule.** Once a Meat task reaches 80% completion, force a context switch to prevent polishing friction on low-return areas.
